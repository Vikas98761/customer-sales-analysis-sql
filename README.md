# Customer Sales Analysis using SQL

## Project Overview
This project analyzes customer purchasing behavior using SQL.

## Objectives
- Calculate total revenue
- Segment customers (High / Medium / Low)
- Identify top selling products
- Analyze city-wise revenue
- Perform monthly growth analysis

## Skills Used
- SQL Joins
- GROUP BY
- CASE WHEN
- Window Functions (LAG, RANK)

## Tools Used
- SQL Server / MySQL
